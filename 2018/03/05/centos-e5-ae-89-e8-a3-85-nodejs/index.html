选择安装版本 稳定版用 curl --silent --location https://rpm.nodesource.com/setup_8.x | sudo bash - 最新版用 curl --silent --location https://rpm.nodesource.com/setup_9.x | sudo bash - 然后执行命令 sudo yum -y install nodejs 再运行代码下载构建工具
