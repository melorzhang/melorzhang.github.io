远程链接redis

Author: Melor Date: 2018-05-09 Updated On: 2018-09-13
Categories: 全栈 配置
Tags:
什么都懂一点，生活更美好一些

  1. 下载redisdesktop并安装工具（若你有其他工具，亦可）;

  2. 修改配置文件 （通过find / -name redis.conf找到文件位置，用vim工具打开修改）

...
bind 127.0.0.1
...
protected-mode yes
...

为

...
# bind 127.0.0.1
...
protected-mode no
...

若需要使用密码,修改

# requirepass foobared

为

requirepass 你的密码
  1. 打开redisdesktop，填写redis所在服务器的ip，即可连接。（需确认服务器开放了6379端口）

补充：在node环境下连接redis 确认上面能正常连接远程redis后，npm install redis,按照说明即可正常连接。
