repeating-linear-gradient 实现虚线边框线

Author: Melor Date: 2017-12-27 Updated On: 2018-09-13
Categories: css 技术分享
Tags:
什么都懂一点，生活更美好一些

话不多说先上图 代码如下：

<!DOCTYPE html>

<html lang="en" xmlns="http://www.w3.org/1999/xhtml">

<head>
    <meta charset="utf-8" />
    <title></title>
    <style>
        .box {
            margin: 80px 30px;
            width: 200px;
            height: 200px;
            position: relative;
            background: #fff;
            float: left;
        }

        .box:before {
            content: '';
            z-index: -1;
            position: absolute;
            width: 220px;
            height: 220px;
            top: -10px;
            left: -10px;
        }

        .mine::before {
            background: repeating-linear-gradient(135deg, red, red 5px, white 5px, white, white 10px, blue 10px, blue, blue 15px, white 15px, white, white 20px);

        }
    </style>
</head>

<body>
    <div class="box mine"></div>
</body>

</html>

重点: background: repeating-linear-gradient(135deg, red, red 5px, white 5px, white, white 10px, blue 10px, blue, blue 15px, white 15px, white, white 20px); 135deg,5px,10px,15px都是可以改的，其中5px也可以用5%，但是整体需要单位一致，然后也必须从小到大写，具体可以参考mdn相关文档
