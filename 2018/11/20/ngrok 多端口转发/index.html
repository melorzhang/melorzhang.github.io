ngrok ERR_NGROK_108 错误‘解决’方案

windows用户，当执行ngrok http 8080 后，若再执行ngrok http 8000，会报错ERR_NGROK_108，免费用户只能执行一个转发，百度，google没找到对应解决方案，于是只能再注册一个新账号，然后获得该账号的authtoken，再执行ngrok http -authtoken=Your_Authtoken 8000 ,就可以转发第二个端口了，如果还有更多端口，可以重复用这个方法，缺点就是要注册多个账号。
